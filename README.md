# Use Libraries

- React-icons : `yarn add react-icons`
- React-Router-Dom : `yarn add react-icons react-router-dom`
- jest-fetch-mock : `yarn add --dev jest jest-fetch-mock`
- Jest : `yarn add jest --dev`

To install all dependencies, run

```shell
yarn install
```
